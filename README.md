🎮 Sobre o jogo

Space Sort é um sort ball puzzle onde o jogador precisa organizar bolas coloridas dentro de tubos, usando lógica e estratégia.

Simples de aprender, difícil de dominar.

🧠 Gameplay
Arraste bolas entre os tubos
Combine cores iguais
Evite movimentos errados
Complete todos os níveis
✨ Features
🎨 Estilo visual minimalista com tema espacial
🧩 Níveis progressivos (cada vez mais difíceis)
🎮 Controles simples (drag & drop)
🔁 Alto fator replay
🧘 Experiência relaxante

🛠️ Tecnologias
Unity (C#)
Sistema de UI custom
Lógica de puzzle baseada em estados
